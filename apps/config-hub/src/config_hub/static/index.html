💡 Jenkins Tips & Tricks for Bot-Only Setup

🔒
Credentials & PAT Rotation

Remember that Personal Access Tokens (PATs) expire! Set a calendar reminder to rotate credentials under Manage Jenkins → Credentials to keep your SCM checkouts healthy.

⚡
Shallow Git Clones

Enabling shallow cloning limits history checkout to depth: 1. This saves substantial agent disk space and speeds up the checkout stage significantly.

🧹
Workspace Cleanups

Ensure the Workspace Cleanup Plugin is installed in Jenkins. Wiping the workspace after runs prevents accumulating tens of gigabytes of Flutter build caches on agent disks.

🚦
Restrict Agent Concurrency

Set the agent's Number of executors to 1 (under Nodes → flutter-agent → Configure) to prevent parallel Gradle runs from choking agent RAM and CPU resources.
